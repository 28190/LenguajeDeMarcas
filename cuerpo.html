Manual HTML y CSS

Cuerpo → 
<body>

La etiqueta cuerpo tiene el contenido del documento, es decir, lo que sí veremos desde el navegador, como texto, imágenes, tables, etc...

Elementos

Los elementos HTML normalmente comienzan con 
<
y acaban con 
>
.
Los elementos son la base del HTML. Podemos distinguir entre varios tipos.

Elemento vacio → 
<img src="Resources/HTML5 logo.png" />

Este tipo de elementos se abren y cierran en una sola etiqueta.
Es opcional escribir el 
/
del final.

Elemento anidado → 
<head></head>

Los elementos pueden contener otros elementos.

Atributos

Los elementos HTML pueden tener atributos, éstos proporcionan información adicional a los elementos. Su sintaxis es: 
nombre_del_atributo="valor"
.

Ejemplo

<a href="html_main.html">Esto es un link</a>

Semántica

Con la nueva versión de HTML5 han surgido nuevas etiquetas con las que trabajar que enriquecen semánticamente nuestro código, en el sentido de que describen lo que contienen al navegador o programador que lea el código.

Hasta ahora en las páginas web se usaban elementos como: 
<div> o <span>

Que no indican nada sobre la información que contienen.

Sin embargo ahora tenemos elementos como: 
<table>, <article>, <nav>, <figure>

Que nos dan una idea más clara de lo que van a contener.

En su lugar hay que usar unas entidades propias de HTML.

Unidades

Muchas de las propiedades usadas de CSS utilizan unidades para determinar su longitud. Las más utilizadas y remendadas son: 
em, px y %

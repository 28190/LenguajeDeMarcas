Cabezera → 
<head>

Este elemento HTML contiene un conjunto de datos que describen el contenido del documento HTML.

Título → 
<title>

Uso

La etiqueta 
<title>
define el título del documento y es necesaria en todos los documentos HTML.
Esta etiqueta proporciona el nombre de la página web al navegador, que es visible en la barra del navegador.

Ejemplo

<!DOCTYPE html>
<html lang=es>
<head>
	<title>Ejemplo - Cabezera>Título</title>
</head>
<body>
	Como puedes ver en la barra del navegador,
	el nombre de este documento HTML es "Ejemplo - Cabezera>Título",
	tal y como lo indica en la cabezera de su código.
</body>
</html>

Metadatos → 
<meta>

Uso

La etiqueta <meta> proporciona datos que indican información sobre el documento y no se muestran en el navegador.
Por ejemplo:
  • La codificación de los caracteres del documento
  • El autor del documento
  • Las palabras clave para aparecer en buscadores web
  • Una pequeña descripción del documento

Ejemplo

<!DOCTYPE html>
<html lang=es>
<head>
	<title>Ejemplo - Cabezera>Metadatos</title>
	<meta charset="UTF-8">
	<meta name="description" content="Ejemplo de metadatos en la cabezera">
	<meta name="keywords" content="HTML,Cabezera,Metadatos">
	<meta name="author" content="Javier Calvo García">
</head>
<body>
	Como puedes ver la página web es exactamente igual que sin los metadatos.
</body>
</html>

CSS

CSS es el leguaje que se utiliza para darle estilo a un documento HTML. Dentro de la cabecera podemos insertar estilos de dos formas distintas.

CSS Interno → 
<style>asdd</style>

Uso
El formato indicado dentro del estilo se aplicará al documento actual, y no se necesita un archivo .css aparte para darle formato al documento HTML.
Ejemplo
<!DOCTYPE html>
<html lang=es>
<head>
	<title>Ejemplo - Cabezera>CSS Interno</title>
	<style>
		body {
			background-color: green;
		}
	</style>
</head>
<body>
	Como puedes ver la página web utiliza el estilo añadido en la cabezera.
</body>
</html>
Ir a ejemplo

CSS Externo → 
<link rel="stylesheet" href="styles.css">

Uso
A través de este método, utilizamos un archivo distinto para darle formato a nuestro documento HTML, pudiendo usar el mismo archivo .css para distintos docuemntos HTML, evitando así repetir el código css en ambos documentos.
Ejemplo
<!DOCTYPE html>
<html lang=es>
<head>
	<title>Ejemplo - Cabezera>CSS Externo</title>
	<link rel="stylesheet" href="Examples\Cabezera.CSSExterno.css">
</head>
<body>
	Como puedes ver la página web utiliza el estilo del documento externo que hemos indicado.
</body>
</html>

body {
	background-color: yellow;
}

Manual HTML y CSS

Aquí explicaremos algunos de los estilos que se encuentran en el CSS.

La estructura es la siguiente:
selector {
	propiedad:valor;
}

Selector

De identificador: Debes usar # antes de la id.
De clase: Debes usar . antes de la clase.

font-family

La sintaxis es la siguiente: 
font-family: family-name|generic-family|initial|inherit;

font-style

Podemos ponerle distintos formatos, como: 
normal, italic, oblique
.

font-weight

Se usa para cambiar el grosor de la letra: 
normal, bold, 900
.

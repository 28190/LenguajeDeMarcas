Manual HTML y CSS

Títulos → 
<h1>...<h6>

En HTML existen unas etiquetas propias que vienen con un formato predefinido que representan títulos.
Están ordenador de forma que el título más general empieza siendo 
<h1>
y acaba siendo el más concreto 
<h6>
.

Tenemos también la etiqueta 
<hgroup>
que se utiliza para agrupar varios títulos.
Dentro de esta etiqueta SOLO puede haber declaraciones de stitulos nada más.

Elementos de formato

Bastardilla 
<i>
vs Énfasis 
<em>

Uso

Visualmente ambos elementos se muestran igual, sin embargo dependiendo de nuestro propósito para darle este formato, debemos usar uno u otro.

<em>
es un nuevo elemento semántico de HTML5, su uso debe ser para marcar con énfasis las partes importantes de un texto.
Sin embargo, 
<i>
simplemente le da el formato al texto, sin dar ningún contexto semántico.

Ejemplo

Las <em>patatas</em> se hacen con <i>ácido sulfúrico</i>.

Las patatas se hacen con ácido sulfúrico.

<b>
vs Importante 
<strong>

Uso

Visualmente ambos elementos se muestran igual, sin embargo dependiendo de nuestro propósito para darle este formato, debemos usar uno u otro.

<strong>
es un nuevo elemento semántico de HTML5, su uso debe ser para marcar con énfasis las partes importantes de un texto.
Sin embargo, 
<b>
simplemente le da el formato al texto, sin dar ningún contexto semántico.

Ejemplo

Las <b>patatas</b> se hacen con <strong>ácido sulfúrico</strong>.

Las patatas se hacen con ácido sulfúrico.

Subrayado 
<u>
vs Insertado 
<ins>

Uso

Visualmente ambos elementos se muestran igual, sin embargo dependiendo de nuestro propósito para darle este formato, debemos usar uno u otro.

<ins>
es un nuevo elemento semántico de HTML5, su uso debe ser para marcar las partes de un texto que han sido añadidos al documento.
Sin embargo, 
<u>
simplemente le da el formato al texto, sin dar ningún contexto semántico.

Ejemplo

Las <u>patatas</u> se hacen con <ins>ácido sulfúrico</ins>.

Las patatas se hacen con ácido sulfúrico.

Borrado 
<del>
vs Incorrecto 
<s>

Uso

Visualmente ambos elementos se muestran igual, sin embargo dependiendo de nuestro propósito para darle este formato, debemos usar uno u otro.

<del>
sirve para identificar texto que se ha eliminado de un documento.
Mientras que 
<s>
se utiliza para textos que son incorrectos, imprecisos o irrelevantes.

Ejemplo

Las <del>patatas</del> se hacen con <s>ácido sulfúrico</s>.

Las patatas se hacen con ácido sulfúrico.

<small>

Este elemento simplemente hace el texto más pequeño. 

<mark>

Este elemento sirve para destacar una parte del texto. 

<sub>

Este elemento sirve para mostrar un texto más bajo y pequeño de lo normal. 

<sup>

Este elemento sirve para mostrar un texto más alto y pequeño de lo normal.

Elementos de líneas

Párrafos → 
<p>

La etiqueta 
<p>Esto es un párrafo</p>
sirve para escribir texto seguido y que cuando acabe se inserte un salto de línea.

Salto de línea → 
<br>

Este elemento inserta un salto de línea allá donde lo pongas.
Hay un elemento que puede ser usado para que en el caso de que al renderizar un documento vaya a poner un salto de línea, lo ponga allí:
<wbr>
.

Línea horizontal → 
<hr>

La etiqueta 
<hr>
se utiliza para diferenciar entre dos temas distintos, visualmente se muestra como una línea.

Texto preformateado → 
<pre>

La etiqueta 
<pre>
se utiliza para insertar texto con un formato parecido a código.
<pre>System.out.println();</pre>

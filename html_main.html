Manual HTML y CSS

¿Qué significa HTML?

Hypertext Markup Language (HTML) es el lenguaje de marcado estandar para páginas web.
Se suele usar en conjunto con Cascading Style Sheets (CSS) y JavaScript (JS).
Este manual está diseñado para la versión HTML5, aunque muchas de las cosas que explicaremos también funcionarán en otras versiones.

Formato del codigo HTML

Lo primero que debemos escribir en nuestro documento HTML es el tipo de documento que es.
<!DOCTYPE html>
En HTML5 no importa si DOCTYPE está en mayúsculas o minúsculas.
Esto indica al navegador como renderizar la página.

Listas

Existen dos tipos de listas, ordenada y no ordenada.
Para añadir elementos a una lista utilizamos las etiquetas 
<li>
.
Las listas tienen las siguiente propiedades en CSS:
  • list-style-image
    : Esta propiedad cambia el marcador antes de cada elemento en una imagen.
    La sintaxis es la siguiente: 
    list-style-image: url("https://i.imgur.com/nvfBX3u.png");

  • list-style-type
    : Esta propiedad cambia el estilo de los marcadores antes de cada elemento.
    • Algunos valores para listas no ordenadas: 
      circle, square disc
      .
    • Algunos valores para listas ordenadas: 
      decimal, lower-roman, lower-latin
      .

  • list-style-position
    : Esta propiedad cambia la posición de los marcadores antes de cada elemento.
    Este comando admite dos valores: 
    outside o inside
    .
    Lo que hace es mover los marcadores dentro (o fuera) fuera del elemento de la lista.

Lista ordenada → 
<ol>

En la declaración de la lista ordenada también puedes especificar:
  • Tipos de marcadores: 
    <ol type="value">
    .
    Lista de valores:
    • 1
    • a
    • A
    • i
    • I

  • Número por el que empezar a contar: 
    <ol start="value">
    .

  • Si debe contar de forma ascendente o descendente: 
    <ol reversed>
    .
<ol start="23" reversed>
	<li>
		Objeto 2.1
		<ol type="a">
			<li>
				Objeto 3.1
			</li>
			<li>
				Objeto 3.2
			</li>
		</ol>
	</li>
	<li>
		Objeto 2.2
	</li>
</ol>

Tablas → 
<table>

En una tabla podemos distintguir las siguientes partes:
  • Encabezado → 
    <thead>
  • Cuerpo → 
    <tbody>
  • Sumario → 
    <tfoot>
  • Leyenda → 
    <caption>

Esta diferenciación es puramente semántica para que el navegador o el que lea el código entienda a que se refiere el contenido de esa etiqueta.

Para añadir los nombres de cada columna a una tabla utilizamos las etiquetas 
<th>
.
Para añadir filas a una tabla utilizamos las etiquetas 
<tr>
.
Para añadir columnas a una tabla utilizamos las etiquetas 
<td>
.
A continuación mostraremos algunas de las propiedades de CSS que se le pueden aplicar a los elementos de las tablas. 
<table id="TablaEjemplo1">
	<thead>
		<tr>
			<th>Asignaturas</th>
			<th>Notas</th>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td>Matemáticas</td>
			<td>6</td>
		</tr>
		<tr>
			<td>Inglés</td>
			<td>7</td>
		</tr>
	</tbody>
	<tfoot>
		<tr>
			<td>Media</td>
			<td>6.5</td>
		</tr>
	</tfoot>
</table>
#TablaEjemplo1, #TablaEjemplo1 td, #TablaEjemplo1 th {
	border: 5px solid blue;
	border-collapse: collapse;
	width: 30%;
}

#TablaEjemplo1 th {
	height: 35px;
	color:	red;
	vertical-align: top;
}

#TablaEjemplo1 th, #TablaEjemplo1 td {
	padding: 15px;
}

#TablaEjemplo1 tfoot td {
	color: green;
	text-align: center;
}

#TablaEjemplo1 tr:hover {
	background-color: #f5f5f5;
}

Enlaces → 
<a>

Para añadir enlaces se utiliza el elemento 
<a href=""></a>

Este texto contiene un enlace a la página de Google.
Además de usar páginas web en línea, puedes usar ficheros locales.
Si quieres ir a una carpeta anterior, deberías usar la siguiente sintaxis 
href="../pagina.html"
.
Si por ejemplo, quieres ir a otra parte de ese mismo documento, entonces tendrás que crear un punto de anclaje y en href indicar su id precedida de un #:
<a href="#test">Click aquí para ir al parrafo con id="test"</a>
<p id="test">Esto es una prueba.<p>

El atributo 
target
indica como quieres abrir el documento: 
  • _blank
    : Abrir en una nueva ventana/pestaña. 

  • _self
    : Abrir en la misma pestaña.
